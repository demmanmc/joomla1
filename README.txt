1- What is this?
	* This is a JoomlaOne package based on Joomla! installation/upgrade package to version 3.x

2- What is JoomlaOne?
	* JoomlaOne is a Joomla derived CMS that allows the use of one Joomla folder with more than one Joomla website.
	* It's a free and OpenSource software, distributed under the GNU General Public License version 2 or later

3- Is JoomlaOne for you?
	* JoomlaOne is mostly the right solution for developers that creates a lot of Joomla websites
